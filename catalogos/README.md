# Catalogos digitales

Estas paginas estan aisladas del sitio principal y pensadas para compartir por link desde GitHub Pages.

## Links esperados

- `/catalogos/`
- `/catalogo-personajes/`
- `/catalogo-contorno/`

## PDFs

Coloca los archivos PDF en:

- `catalogos/pdfs/personajes.pdf`
- `catalogos/pdfs/contorno.pdf`

Los visores usan PDF.js para renderizar las paginas y StPageFlip para la experiencia de flipbook.

## Notas

Estas paginas no estan enlazadas desde el menu principal del sitio mientras se terminan de disenar.
